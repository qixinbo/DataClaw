# DataClaw

Data Analysis Platform.

## Structure

- `backend/`: FastAPI backend
- `frontend/`: React frontend (Vite + TailwindCSS + Shadcn UI)
- `nanobot/`: Core AI agent framework

## Setup

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
